# bingo_diferencas

## Descrição
O arquivo bingo_diferencas.c implementa um programa que simula um jogo de Bingo, calculando a diferença absoluta entre as bolas sorteadas. O programa recebe entradas de pares de números representando os sorteios de bolas até que a linha "0 0" seja fornecida, indicando o fim da execução.

## Objetivo
O principal objetivo do código é calcular a diferença absoluta entre dois números (bolas sorteadas) e exibir o resultado até a entrada de "0 0", que finaliza o jogo.

## Como Funciona o Código

1. O programa lê os pares de números fornecidos pelo usuário.
2. Para cada par, calcula a diferença absoluta entre os dois números.
3. Exibe a diferença na tela.
4. Repete o processo até encontrar a entrada "0 0", que encerra a execução.
